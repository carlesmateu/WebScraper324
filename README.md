# WebScraper324
Repositori que conté un script de web scraping desenvolupat en Python per extreure notícies del portal 324.cat. El projecte recorre diverses seccions, extreu informació rellevant de cada article i genera un fitxer CSV amb el dataset final.lcula atributs derivats com el dia de la setmana per facilitar l’anàlisi del dataset resultant.
